# cinema-mobile-app
React Native App for Ticket Booking

## Installation

```bash
git clone https://github.com/lv999CaNoc/cinema-app.git
```

Install node modules with npm
```bash
cd cinema-mobile-app
npm install
```

## Running
```bash
npm start
```
